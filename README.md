# RouteNexus 🚚⚡
**Smart Supply Chain Disruption Detection**

RouteNexus is an AI-driven logistics dashboard designed to preemptively detect supply chain disruptions and recommend optimized route adjustments in real-time. 

### 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Python, Flask, Flask-CORS
* **AI Integration:** Architecture designed for Google Gemini API (currently utilizing a simulated mock-API backend for the prototype demo due to hackathon free-tier quota limits).

### 🚀 How to Run Locally
1. Clone this repository to your local machine.
2. Ensure you have Python installed.
3. Install the required dependencies:
   `pip install flask flask-cors`
4. Run the backend server:
   `python app.py`
5. Open `index.html` in your web browser to view the live dashboard.

### 💡 Core Features
* **Live Monitoring:** View current conditions of active shipments.
* **Preemptive AI Alerts:** Detects severe weather and traffic conditions before they impact delivery timelines.
* **Dynamic Re-routing:** Suggests alternative paths to bypass localized bottlenecks.
